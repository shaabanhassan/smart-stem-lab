⚙️ Developer Setup — Google Client ID
How to get your Client ID in 5 steps:
  1. Go to console.cloud.google.com
  2. Create a project → Enable Google Identity API
  3. APIs & Services → Credentials → Create OAuth 2.0 Client ID
  4. Type: Web application → In Authorized JavaScript Origins add your exact URL (e.g. http://localhost:8000)
  5. Copy the Client ID → paste above → Save & Activate ✓
⚠️ Important: Google Sign-In will NOT work if you open this file directly in the browser (file://). You must serve it over HTTP. Quickest way:
python -m http.server 8000
Then open: http://localhost:8000/grade4_portal.html
